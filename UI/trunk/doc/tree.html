树形菜单组件文档

模块加载名称：flyertree

全部参数

语法：flyerui.tree(settings)

参数 描述
settings 部分必填，部分可选。用于配置flyerpage的键值对集合
键 默认值 类型 描述
name Tree String 树形菜单节点显示的文本
spread false Object 树形菜单节点显示的文本
href null String 树形菜单节点要跳转的链接地址
target _blank String 树形菜单节点跳转地址的方式,等同于A标签的target设置
childrens null Array 同data节点，可无限延伸
使用案例 

/*html部分*/




/*js部分*/

        });
