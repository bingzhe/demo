模块加载名称：flyerComboBox

使用说明
声明一个div容器盒子，调用flyer.combobox($dom,{}),第一个参数传入容器盒子jQuery对象，第二个传入属性配置,如下:

 //申明容器盒子
属性使用说明
键 默认值 类型 描述
isMulti true Boolean 是否支持多选.
disabled false Boolean 是否禁用下拉
allowSearch true Boolean 是否允许索引内容
multipleSeparator ; String 显示值分隔的符号
fieldKey id String 要绑定的键
fieldValue value String 要绑定的值,默认值为
placeholder true String 显示在下拉框上的 placeholder
searchPlaceholder 在这里可以快捷搜索到你想找的.. String 搜索的框里的placeholder
url 空 String 请求服务数据源地址
data [] Array JSON数据Data ,优先于请求的服务数据源地址
selectAll true Boolean 是否可以选中全部
allowSelectAll true Boolean 点击全选是否选中所有的值
fnDataProcessing function() { this.data = this.data.rows; }, Function 开放一个数据结构处理的方法
fnSelected function() { return false; } Function 选中事件
fnBeforeSelected function() { //废弃后的事件... } Function 选中之前事件
fnAfterSelected function() { //废弃后的事件... } Function 选中之后事件
结语

你是否发现，主题、尺寸、图标、圆角的交叉组合，可以形成难以计算的按钮种类。 另外，你可能最关注的是配色，flyer内置的六种主题的按钮颜色都是业界常用的标准配色，如果他们仍然无法与你的网站契合， 那么请先允许我“噗”一声。。。然后你就大胆地自撸一个颜色吧！比如：粉红色或者菊花色（一个有味道的颜色）
