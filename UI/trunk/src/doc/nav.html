导航相关 - 页面元素

导航一般指页面引导性频道集合，多以菜单的形式呈现，可应用于头部和侧边，是整个网页画龙点晴般的存在。面包屑结构简单，支持自定义分隔符.

水平、垂直、侧边三个导航的HTML结构是完全一样的，不同的是：

垂直导航需要追加class：flyer-nav-tree

侧边导航需要追加class：flyer-nav-tree flyer-nav-side

面包屑

首页 国际新闻 亚太地区 正文

    <a href="">首页</a>
    <a href="">国际新闻</a>

你还可以通过设置class名称flyer-separator-line来自定义分隔符。如：

首页 国际新闻 亚太地区 正文 

<span class="flyer-breadcrumb flyer-separator-line">
    <a href="">首页</a>
    <a href="">国际新闻</a>
    <a href="">亚太地区</a>
    <a><cite>正文</cite></a>
</span>

当然，你还可以作为小导航来用，如：
