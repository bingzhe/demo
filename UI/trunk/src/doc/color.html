场景色

  • #F7B824

    暖色系，一般用于提示性元素

  • #FF5722

    比较引人注意的颜色

  • #01AAED

    用于文字着色，如链接文本

  • #2F4056

    侧边或底部普遍采用的颜色

Flyerui并非不敢去尝试一些亮丽的颜色，但许多情况下一个它可能并不是那么合适， 所以我们把这些颜色归为“场景色”，即按照实际场景来呈现对应的颜色， 比如你想给人予警觉感，可以尝试用上面的红色。他们一般会出现在Flyerui的按钮、 提示性元素、及一些侧边元素上。
