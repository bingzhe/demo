语法：flyerui.pager(settings)

键 默认值 类型 描述
elem flyer-upload-file String 指定元素的选择器，默认直接查找class为layui-upload-file的元素
text 空字符串 String 按钮显示的文本
url 空字符串 String 上传文件的地址
method post String 设置http类型，如：post、get
enctype avatar String 上传控件的名称
success function() { return false; } Function 上传成功后的事件
