检索框组件文档

模块加载名称：flyer-ui-searches

属性使用说明
键 默认值 类型 描述
isMulti true Boolean 是否允许多个关键词查询，默认是允许的
url 空 String 请求数据的地址
searchName 空 String 对应的 Search Name名字
fieldValue 空 String 对应的数据主键值
fieldName 空 String 显示的文本值
requestData [] Array 开放请求数据的方法
spaceTime 1000 Int 事件响应的间隔时间,默认1秒(设置值为毫秒)
separtor ; String 分隔符,默认;
distinck true Boolean 是否去重
emptyMsg 未检索到相关的数据.. String 数据为空时的提示语句
fnDistinckTips function() { console.log("该项已经选中...."); } Function 是否去重相关的提示方法
fnDataProcessing function() { this._data = this._data.rows; } Function 开放一个数据结构处理的方法
fnSelected function() { return false; } Function 选中之后的方法
内置方法使用说明
方法名 参数 返回值 描述
getSelectedData 空 JSON 获取选中的值,返回的数据格式为:[ { fieldValue: "", fieldName: "" },{...}]

你是否发现，主题、尺寸、图标、圆角的交叉组合，可以形成难以计算的按钮种类。 另外，你可能最关注的是配色，flyer内置的六种主题的按钮颜色都是业界常用的标准配色，如果他们仍然无法与你的网站契合， 那么请先允许我“噗”一声。。。然后你就大胆地自撸一个颜色吧！比如：粉红色或者菊花色（一个有味道的颜色）
