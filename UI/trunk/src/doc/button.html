按钮 - 页面元素

向任意HTML元素设定class="flyer-btn"，建立一个基础按钮。通过追加格式为flyer-btn-{type}的class来定义其它按钮风格。内置的按钮class可以进行任意组合，从而形成更多种按钮风格。
